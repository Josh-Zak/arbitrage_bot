# Cryptotrage
https://cryptotrage.vercel.app/

## About
- Passion project started upon discovering price discrepancy between exchanges and taking advantage to turn a profit.
- Developed with Vue.js for the frontend and Node.js using express for the backend.
- Hosted using Vercel for the frontend and Render for the backend (Shuts down on idle. Takes ~30s to startup).

## How to Run Locally
### Frontend
    cd frontend
    npm run dev

### Backend
    cd backend
    node index.js or nodemon index.js
